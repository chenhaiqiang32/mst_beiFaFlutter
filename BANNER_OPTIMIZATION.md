## 🎯 响应式设计特性

### 自适应尺寸
```dart
final screenWidth = MediaQuery.of(context).size.width;
final bannerHeight = screenWidth * 0.5; // 高度为屏幕宽度的50%
```

### 文字定位（百分比）
```dart
Positioned(
  left: screenWidth * 0.05,  // 左边距为屏幕宽度的5%
  top: bannerHeight * 0.25,  // 顶部位置为横幅高度的25%
  right: screenWidth * 0.4,  // 右边距为屏幕宽度的40%
  // ...
)
```

### 自适应字体大小
```dart
TextStyle(
  fontSize: screenWidth * 0.06,    // 主标题：屏幕宽度的6%
  fontSize: screenWidth * 0.035,   // 副标题：屏幕宽度的3.5%
)
```

## 🔧 技术实现

### 布局结构
```dart
Container(
  height: bannerHeight, // 自适应高度
  child: ClipRRect(
    child: Stack(
      children: [
        // 背景图片

        // 文字内容
        Positioned(
          left: screenWidth * 0.05,
          top: bannerHeight * 0.25,
          right: screenWidth * 0.4,
          child: Column(...),

      ],
    ),
  ),
```
